# go-music

## Getting Started
### MySQL (Database)
- Install MySQL
- Create [database and tables](sql/create_schema.sql)

### Gin (Backend RESTful API Server)
- Install Golang
- Install Gin
- Start Gin Server

### React (Frontend Server)
- Install [Node.js](https://nodejs.org/en/)
- Install all dependencies.
  ```bash
  cd go-music
  npm install
  ```
- Start react server.
  ```bash
  npm start
  ```
